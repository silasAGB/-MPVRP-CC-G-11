Ce code constitue une base d'expérimentation pour résoudre les problèmes de routage incohérent. Bien qu'il ne soit pas encore garanti fonctionnel sur 100% des instances,il introduit :

-Élimination des sous-tours (MTZ) : Tentative de suppression des boucles isolées entre stations en imposant un ordre de passage mathématique.
-Contrainte de Garage : Verrouillage du flux pour forcer chaque camion à démarrer et finir à son point d'attache réel.
